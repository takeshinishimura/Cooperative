棒グラフ

ここでは，農業協同組合の部門別損益の把握を試みる。

まず，令和2事業年度の総合農協統計表の損益計算書の統計データがどこにあるかを確認する。

農業協同組合及び同連合会一斉調査 確報 総合農協統計表　令和２事業年度総合農協統計表　第１部　都道府県別統計表　損益計算書

https://www.e-stat.go.jp/index.php/stat-search/database?statdisp_id=0001998613

e-Statには部門別損益のデータセットも用意されており，次のページから確認できる。

農業協同組合及び同連合会一斉調査 確報 総合農協統計表　令和２事業年度総合農協統計表　第１部　都道府県別統計表　部門別損益

https://www.e-stat.go.jp/index.php/stat-search/database?statdisp_id=0001998643

部門別損益の表は，損益計算書を組み替えたものであると考えられる。 しかしながら，各統計表の数値は一致しない。 例えば，事業総利益は前者では1,710,097,906，後者では1,707,803,082となっている（単位はいずれも千円）。 本来，これらの数値は一致するはずであることから，どちらかが再集計しているのかもしれない。

上記2つのデータセットを提供するページのいずれにもExcelファイルがないことから，どこかにExcelファイルを提供するページがあるはずである。 探すと，次の2つのページが見つかる。

農業協同組合及び同連合会一斉調査 確報 総合農協統計表　令和２事業年度総合農協統計表　第１部　都道府県別統計表　損益計算書

https://www.e-stat.go.jp/stat-search/files?stat_infid=000040041555

農業協同組合及び同連合会一斉調査 確報 総合農協統計表　令和２事業年度総合農協統計表　第１部　都道府県別統計表　部門別損益

https://www.e-stat.go.jp/stat-search/files?stat_infid=000040041636

これらのページからExcelファイルをダウンロードして比較してみると，上の2つのデータセットと同じ数値であり，損益計算書と部門別損益の違いも同じである。 ただし，Excelファイルの方が情報が多いことが分かる。 集計組合数が前者は587であるのに対して，後者は582である。 後者が5つ少ない。 この理由が後者の表の下の注に以下のように記されている。 「農林中央金庫及び特定農水産業協同組合等による信用事業の再編及び強化に関する法律（平成八年法律第百十八号）第42条第1項に基づき信用事業譲渡を行い業務の代理を行う農協は含まない。」 これらのことから，e-StatからAPIでデータを入手する場合，同じデータを含むExcelファイルを確認することも怠らない方がよいと言えよう。

ここでは，部門別損益のデータセットを用いて図を作成することにする。

1 e-Statからデータ取得

早速，次のコードを実行しよう。 次のコードにある statsDataId = "0001998643" の数字はアクセスしたい統計データのIDである。

library(httr)

response <- httr::GET(
  url = "https://api.e-stat.go.jp/rest/2.1/app/getSimpleStatsData",
  query = list(
    appId = keyring::key_get("e-stat"),
    statsDataId = "0001998643",
    sectionHeaderFlg = 2
  )
)
response <- httr::content(response)
df <- read.csv(text = sub("^.*\\\"VALUE\\\"\n", "", response), header = TRUE, check.names = FALSE)

2 棒グラフ

まず，税引前当期利益を比較してみよう。

df1 <- df[df[, 2] == "税引前当期利益" & df$`(R001-02-1-050)部門別損益計算書（総括版）` != "計" & df$value != "-", ]

library(ggplot2)
library(ggsci)

g <- ggplot(data = df1) +
  geom_bar(aes(x = `(R001-02-1-050)部門別損益計算書（総括版）`,
               y = value,
               fill = `(R001-02-1-050)部門別損益計算書（総括版）`),
           stat = "identity") +
  theme_bw() +
  scale_x_discrete(limits = rev) +
  theme(legend.position = "none") +
  scale_fill_nejm() +
  theme(axis.title = element_blank())
g + theme(text = element_text(family = "HiraKakuProN-W3"))

注意

R（RStudio）で実行（QuartoをRender）した場合とVisual Studio CodeでQuartoをRenderした場合とで，縦軸の順番が異なる。 後で実行する，levels(factor(df$(R001-02-1-050)部門別損益計算書（総括版）))の結果にも違いが生じる。 おそらく後者の処理の中でバグが生じている可能性が考えられる。 ここに表示されるものは後者でRenderした結果である。

営農指導事業，生活その他事業，農業関連事業の税引前当期利益がマイナスであることが分かる。 これらについて詳細を確認するために，次に，事業総利益と事業管理費のグラフを描いてみよう。

df2 <- df[str_detect(df$`(R001-02-2-011)区分`, "^(事業総利益|事業管理費)$") &
